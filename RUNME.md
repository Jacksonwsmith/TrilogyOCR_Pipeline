# RUNME

Quickest way to run the project locally.

## 1. Start the app

```bash
./run_web.sh
```

## 2. Open the UI

Use `http://localhost:8080`.

## 3. Run extraction

- Upload one or more PDF files
- Click **Run Extraction**
- Watch live progress/timing
- Download CSV or open CSV preview

## 4. Stop the app

In terminal: `Ctrl+C`

---

## Notes

- Keep your API key in `.env` as:
```env
MISTRAL_API_KEY=your_real_key_here
```
- If the browser shows stale styles, hard refresh: `Cmd+Shift+R`
